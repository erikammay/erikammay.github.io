HTML & CSS

Most people have used websites before that use HTML and CSS. They are what I would consider, the two most important and basic coding languages used to make websites. HTML is used to build the structure, and CSS is for styling. Without any CSS, the website would be absolutely terrible; it will not be easy to navigate, confusing, and ugly.


HTML is the absolute basic language to code a website. It is the first thing you do, and the main components of the website are from HTML. An HTML tag looks something like, "<>," this. If there were the letter, "img" inside the brackets, it would be an image tag in HTML. It is used to display an image. A tag is a word or letters with an angle bracket on the left and another on the right. They tell the computer what is supposed to be displayed. “<>,” with an h1 inside is a heading tag.

If you have ever put your credit card information, phone number, or any other personal information, chances are, the website has an SSL certificate. SSL certificates allow a website to use SSL or TLS.



CSS is used to design a website. It has something called rules, which has selectors in it. Selectors take the tag from HTML, and add styling to whatever the element is.
